Brenda opens Navilum on iPhone

iOS prod build installed via TestFlight. Supabase session auto-restores. Home dashboard loads adherence ring + today's doses.

TestFlight pending ASC record

Scans her chemo bottle

Points camera at pill bottle → Azure CV OCR reads label → ScanResultModal pre-fills medication name and dosage.

Selects "21 on / 7 off" chemo cycle

Native FrequencyCode for oncology cycles. Schedule auto-generates rest periods. This is Patent #2.

Adds her husband as caregiver

Family screen → add caregiver → SMS opt-in confirmation. Caregiver gets a token-gated dashboard URL — no app install required.

LIVE

Dose time arrives — escalation begins

Notification fires. Wake-screen with voice-clone alarm in husband's voice. If unacknowledged in N minutes, refires at escalating cadence. This is Patent #1.

Brenda taps "Taken"

Optional Gemini visual dose verify: snap pill cup → vision API confirms it matches schedule. Adherence ring updates.

LIVE
7

Missed dose path (the killer demo)

Brenda is asleep / in chemo brain. After N escalations, caregiver swarm fan-out fires: husband + sister + daughter all get push + SMS within seconds. Husband opens dashboard URL, sees the missed dose, calls her.

LIVE
8

Side-effect logging closes the loop

Brenda logs nausea. Side-effect correlation analyzer surfaces a pattern: nausea spikes within 2hrs of her morning chemo dose. Adherence health-score panel reflects the week.

Business layer

Market, Financials, Differentials, Marketing

All metrics traceable. Numbers without sources have been removed per the no-fabricated-numbers rule.

Market wedge

Primary: oncology patients + caregivers (53M US caregivers per AARP/NAC 2020; ~2M new US cancer cases/yr per ACS 2024).

Expansion: chronic-illness adults, polypharmacy seniors (~73M US adults 65+ by 2030, US Census), care coordinators.

Differentials vs. incumbents

MyTherapy, Medisafe, Round Health, Apple Reminders: none escalate, none schedule chemo cycles natively, none fan out to a caregiver swarm. We do all three. See deliverables/navilum-validation.html for full landscape.

Revenue model

B2C: Freemium → $9.99/mo premium (caregiver network, escalation rules, voice-clone alarm).
B2B: Oncology clinic + care coordinator dashboard subscription.
Pharma: Adherence/side-effect insights (consent-based) — future.

Unit economics (model)

Infrastructure marginal cost ~$0 at scale (Supabase free tier through ~10K users; Azure CV per-call; Gemini Vision per-call). Premium price $9.99/mo. Detailed model: deliverables/financial-dashboard.html.

GTM

Wedge through oncology clinic partnerships (clinic-assigns-app model). Caregiver-led acquisition via the no-install dashboard URL — caregivers become referrers. Content/SEO around chemo schedule complexity.

Patent moat

Two provisional patents in flight (escalation engine + chemo cycle scheduling). Filing target Jun 30. Reference patent: US 11,663,415 (voice-based digital assistant in healthcare). See platform-patent-specification.html.
